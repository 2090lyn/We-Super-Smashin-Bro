# 💖 We Super Smashin’, Bro

  A 2D pixel-art fighting mini-game built with JavaScript and HTML Canvas.
  
  Two players enter.
  One player leaves victorious.
  The other gets absolutely rizzed.

# 🛠️ Built With

  JavaScript (Vanilla)
  
  HTML5 Canvas API
  
  CSS
  
  Custom pixel art assets
  
  Custom sound integration
